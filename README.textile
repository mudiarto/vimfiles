h1. Usage

In your home directory (Linux or Mac) or 
if you're using Windows, C:\Documents and Settings\Your_User

<pre><code>
git clone git://github.com/kevincolyar/vimfiles.git .vim
</code></pre>
Clone this repo into your home directory either as .vim (linux) or vimfiles (MF
Windows).

Setup
<pre><code>
rake
</code></pre>
